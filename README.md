
# Record Visualiser

A simple WIP visualiser for playing records of your favorite albums. Connects directly to the Spotify API to allow you to customise the scene and add any album you want. You can display them, store them, view them, as well as play them through the spotify app with normal functionality such as skipping, pausing etc.
#
This project started as some practice for Godot over the summer between university years, as well as an introduction to dealing with 3D physics not just in Godot but in 3d graphics in general. Even though it started as just a learning experience, It has turned into a passion project of mine, as not only am I learning heaps about game development and Godot, I get to do it in an environment where I am genuinely interested and excited to see the results because I love music so much. It has also helped me learn a lot about full stack development, as it gave me practical knowledge in commercial scale API calls and how the backend vs the frontend should and does work.
#
Features:
- [x] Scene containing record player
- [x] Basic Physics for moving records, disks, and player
- [x] Playing music from system files
- [x] Basic Animations for records, disks, and player
- [ ] Realistic / Stylized Shading and lighting 
      Possible multi-choice shader system, eg dull, shiny, reflective for when user
      input into the scene is added
- [x] Better scene layout for better user experience
- [x] Improved physics for more user agency and realism
- [x] Play music using Spotify (Or other) API for easier use
- [x] User customizability of scene
