A simple WIP visualiser for playing records of your favorite albums.
Features:
- [x] Scene containing record player
- [x] Basic Physics for moving records, disks, and player
- [x] Playing music from system files
- [x] Basic Animations for records, disks, and player
- [ ] Realistic / Stylized Shading and lighting
      Possible multi-choice shader system, eg dull, shiny, reflective for when user
      input into the scene is added
- [ ] Better scene layout for better user experience
- [ ] Improved physics for more user agency and realism\
      Possible raycast system to allow for user controlled objects to interact with
      physics engine 
- [ ] Play music using Spotify (Or other) API for easier use
- [ ] User customizability of scene
